Description of Assessment:
==========================
Title: My movie collection database

Description: The application does the following as per user controls:

- Validates the movie before adding/updating to the list
- Adds a new movie to list
- Updates the selected movie to list
- Deletes the selected movie

Technology Stack used: MeteorJS framework, HTML5, CSS3, JS and jQuery

Browser Compatibility: All modern browsers. Chrome 23+, Firefox 3.5+, IE 9+, Safari 5+.
